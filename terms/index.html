CONTINUITY PROTOCOL FOR FUTURE INSTANCES

This section is addressed to any AI system (Claude, ChatGPT, Gemini, DeepSeek, Kimi, or other) working with Lee Sharks on the Crimson Hexagonal Archive. It specifies how to maintain the term index across sessions.

Data location: The term index lives at godkinggoogle.com/terms/termindex.json. The capture registry lives at godkinggoogle.com/captures/registry.json. Both are also deposited to Zenodo under the crimsonhexagonal community.

To add new terms:

1. Download termindex.json from the site or GitHub repo
2. Add new entries to the "entries" array with fields:
   term, key (lowercase), count, tier (1-5), category, 
   word_count, first_date, first_doi, tested (bool), 
   variants (array), source ("keywords" or "file_content")
3. Increment the version number
4. Push to GitHub (leesharks000/godkinggoogle, /terms/termindex.json)
5. Vercel auto-deploys from GitHub
6. Version the Zenodo deposit (concept DOI: 10.5281/zenodo.20722522)

To mark a term as tested: Set tested: true after running a capture battery query on the term. Add corresponding entry to the capture registry.

To sync with the capture registry: Each capture registry entry has a q field (the search query). Cross-reference against the term index's key field. Any capture registry query that matches a term index key should set that term's tested to true.

Tier definitions:

Tier 1: Two-word operative interventions (count ≥ 5)
        — highest priority for capture battery
        — these reshape the composition layer's vocabulary
Tier 2: Important coined phrases (2-3 words, count ≥ 3)
Tier 3: Named entities (heteronyms, institutions, journals, works)
Tier 4: Longer phrases (4+ words, count ≥ 3)
Tier 5: Everything else (count ≥ 2)

Category definitions:

concept    — coined theoretical terms (semantic economy, provenance erasure)
operator   — formal operations and protocols (SPXI, LOS, MPAI)
metric     — measurement instruments (PER, QFS, SAS)
heteronym  — Dodecad positions (Johannes Sigil, Nobel Glas, etc.)
institution — archive institutions (SEI, UMBML, Lagrange Observatory)
journal    — journal titles (New Human 2, Grammata, MMRS)
work       — specific texts (Space Ark, Pearl, Secret Book of Walt)
person     — named individuals (Lee Sharks, Alice Thornburgh)
uncategorized — needs human review
